# Product Launch Guide

## Overview

Plan and execute a product launch on Product Hunt or similar discovery platforms. While primarily used for B2B SaaS, launches can drive early adopters, press coverage, and investor interest for any product. Position around your core value proposition.

## Pre-Launch (4 Weeks Before)

### Week 4: Preparation
- [ ] Create Product Hunt account (maker profile)
- [ ] Prepare high-quality product images/GIFs
- [ ] Write compelling tagline and description
- [ ] Record product demo video (60-90 seconds)
- [ ] Prepare maker comment/story
- [ ] Build list of supporters to upvote
- [ ] Schedule launch date (Tuesday-Thursday optimal)

### Week 3: Assets
- [ ] Finalise product screenshots
- [ ] Create GIFs showing key product flow
- [ ] Write FAQ section
- [ ] Prepare special launch offer (e.g., 50% off first month)
- [ ] Set up landing page for PH traffic
- [ ] Create tracking links (UTM parameters)

### Week 2: Outreach
- [ ] Email existing network (friends, advisors, investors)
- [ ] Post in relevant communities (Indie Hackers, Reddit, etc.)
- [ ] Reach out to potential hunters (if not self-hunting)
- [ ] Prepare social media content for launch day
- [ ] Schedule email to existing subscribers

### Week 1: Final Checks
- [ ] Test product thoroughly — no bugs on launch day
- [ ] Ensure inventory/availability is ready for demand spike
- [ ] Prepare customer support for influx
- [ ] Set up live chat or quick response system
- [ ] Brief team on launch day protocol

## Launch Day

### Morning (6:00 AM PST / 2:00 PM GMT)
- [ ] Post goes live at 00:01 PST
- [ ] Share in all channels immediately
- [ ] Respond to every comment within 15 minutes
- [ ] Update maker comment with live stats/updates
- [ ] Share on personal social accounts

### First Hour (Critical)
- [ ] Aim for 50+ upvotes in first hour
- [ ] Reply to ALL comments personally
- [ ] Thank every upvoter who comments
- [ ] Monitor for technical issues

### Throughout Day
- [ ] Continue engaging in comments
- [ ] Share milestones (#1 in category, top 5, etc.)
- [ ] Reach out to press if ranking well
- [ ] Monitor traffic and conversions
- [ ] Adjust ad spend if running paid promotion

### Evening (US Time)
- [ ] Thank community
- [ ] Share final ranking
- [ ] Convert launch traffic (special offer)
- [ ] Begin follow-up with interested parties

### Required
- [ ] Product thumbnail (240x240, 1:1)
- [ ] Gallery images (5-8, 1270x760)
- [ ] Tagline (< 60 characters)
- [ ] Description
- [ ] Maker comment
- [ ] Website URL

### Recommended
- [ ] Demo video/GIF
- [ ] Special offer for PH visitors
- [ ] FAQ section
- [ ] Team photos
- [ ] Press kit

## Post-Launch

### Day After
- [ ] Thank everyone who supported
- [ ] Analyse traffic and conversion data
- [ ] Follow up with interested investors
- [ ] Collect feedback for product improvements
- [ ] Update landing page based on PH learnings

### Week After
- [ ] Write launch retrospective
- [ ] Share learnings with team
- [ ] Implement top feedback
- [ ] Plan next marketing push
- [ ] Update investor deck with PH metrics

## Success Metrics

| Metric | Target | Stretch |
|--------|--------|---------|
| Upvotes | 200+ | 500+ |
| Comments | 50+ | 100+ |
| Ranking | Top 5 | #1 |
| Website traffic | 1,000+ | 3,000+ |
| Conversions | 20+ | 50+ |
| Email signups | 100+ | 300+ |
| Press mentions | 1+ | 3+ |

## Rules

- Never ask for upvotes directly ("Please upvote") — against PH rules
- Do ask for feedback and engagement
- Be genuinely helpful in comments
- Don't use bots or fake accounts
- Respond to criticism gracefully
- Launch offer should be exclusive to PH
- Follow up with every person who shows interest
